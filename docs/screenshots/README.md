# README screenshots

Place static captures here for the root [README.md](../../README.md) **Screenshots** section.

**Naming (recommended)**

- `dashboard.png` — main signed-in dashboard  
- `connect.png` — connect / bulk URLs  
- `leaderboard.png` — leaderboard view  
- `landing.png` — optional marketing hero  

Use **PNG** or **WebP**, ~1280px wide max for reasonable file size. Avoid screenshots with private emails or tokens.
